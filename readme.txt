Adds Keystroke Enabled Login Functionality

== Description ==

Hidden Login provides easy and discrete access to the admin functionality of your site.  Pressing the set keystroke combination will activate Hidden Login, either displaying a modal login window or redirected to the admin area if you are already logged in.  Visit the Hidden Login Options to set what pages are enabled (All Pages or Homepage Only) and set your desired activation keystroke combination.

== Installation ==

1. Download the hidden_login.zip file, unzip and upload the whole directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit the plugin options to set the active pages and activation keystroke combination.
